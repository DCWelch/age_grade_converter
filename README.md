# Age Grade Converter
Web-based tool to calculate the Alan Jones / WMA Age Grade and age-grade-equivelants based off a time, gender, and distance.

Live at:  https://dcwelch.io/agegrading

## Data source
Age grading standards are derived from the Age Grade Tables published by Alan Lytton Jones and adopted by WMA:

https://github.com/AlanLyttonJones/Age-Grade-Tables

This project stores converted JSON versions of the relevant road tables for fast client-side use.
